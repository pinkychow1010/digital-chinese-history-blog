This site focuses on Python and aims to support users to conduct academic research in the humanities using open source programming languages. This is created at the end of 2021 by [Ka Hei](https://www.linkedin.com/in/ka-hei-chow-231345188/) who is a Hongkonger passionated about machine learning and data science. ![]({{site.baseurl}}/images/landscape.jpg " ") The establishment of this site aims to improve accessibility of emerging digital Chinese history resources to programming and data science novels who wish to integrate digital tools with historical research. Nonetheless, most of the content and example also fit for all kinds of qualitative humanities data.
